# 🌍 AQI & HCHO Hotspot Detection

AI-powered Air Quality Index (AQI) and Formaldehyde (HCHO) hotspot detection using satellite data.

**Project for:** ISRO Bharatiya Antariksh Hackathon 2026 (Challenge 03)

## 🎯 Objective

Detect air quality issues and HCHO pollution hotspots across India using real-time Sentinel-5P satellite data.

## 🏗️ Tech Stack

- **Backend:** FastAPI, Python, Google Earth Engine
- **Frontend:** React, Leaflet.js, Tailwind CSS
- **ML:** Scikit-learn (Random Forest, K-Means)
- **Database:** PostgreSQL
